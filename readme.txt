Facebook Profile Master gives you full control over all Facebook User Profile plugins, Facebook User Badges and Facebook Follow Button.


== Description ==

Clean and fast html5 plugin that will make your wordpress "rock". Facebook Profile Master allows you to display all Facebook User Badges and Follow Button.

The cool Facebook Profile Badge you can share your facebook information. The Facebook Photo Badge allows you to share your facebook user photos. The Facebook Like Badge allows you to show your favorite pages. The Facebook Page Badge allows you to share your facebook page. The Facebook Follow Button lets other people follow or subscribe your public updates.

Intended for Facebook User Profiles that need professional integration into Wordpress. Specially fitted for all sorts of artists (Designers, Musicians, Painters, Sculptors, Freelancers, etc.), basically any professional activity that is connected to your personal name and facebook profile page.

The plugin uses the new Facebook API and is built-in html5 and iframe. Once again and typical of TechGasp, the plugin uses NO Javascript, jQuery or Ajax making it free of conflicts or errors.

Main Features: The viral plugin for facebook user profiles - Asynchronous Load - Built-in html5- No Javascript - Profile Badge - Photo Badge - Like Badge - Page Badge - Follow Button- Shortcode Framework ready, to publish inside pages and posts

IMPORTANT: Errors and conflicts free. NO JAVASCRIPT or AJAX

== Installation ==

1. Upload everything into the "/wp-content/plugins/" directory of your WordPress site.
2. Activate in the "Plugins" admin panel.
3. Visit the "Widget Page | Facebook Profile Master" to configure the widget and template widget position.
